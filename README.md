# 🎬 Movie Recommendation System

This is a Machine Learning-based Movie Recommendation System. The system recommends movies based on similarity between movie features using cosine similarity.

---

## 🚀 Project Overview

The goal of this project is to build a recommendation system that suggests movies similar to a selected movie. It uses content-based filtering and vector similarity techniques.

---

## 🧠 Machine Learning Concept Used

- Content-Based Filtering
- Cosine Similarity
- Feature Engineering (genres, keywords, cast, crew, etc.)
- Text Vectorization (CountVectorizer / TF-IDF)

---


## ⚙️ How It Works

1. Data preprocessing and cleaning
2. Feature extraction (combine important columns into one)
3. Convert text data into vectors
4. Compute similarity using cosine similarity
5. Recommend top similar movies

---

## 📊 Input & Output

### Input:
- Movie name selected by user

### Output:
- List of recommended movies similar to the selected movie
